How do I USE the loops ? In other words, how do I install loops into GarageBand?

So you've downloaded loops from this site, or another site, and you understandably want to use them in GarageBand!

(*Note - If you are using Fruity Loops or any other software, then these instructions do not apply. I'll write instructions for Fruity Loops at another time.)

Follow these steps to install loops into GarageBand:

  1. Locate the little Eye Icon to open GarageBand's Loop Browser.

  2. Click the Eye Icon so it turns blue. You should now see your Loop Browser.

  3. Navigate to your folder that contains your new loops. Can't find that folder? Try downloading the loops again and notice WHERE your directory automatically saves them. Odds have it, it's in the "Downloads" folder which is inside the User folder.
